Brief project summary (think WPD1 introduction). You can use this to augment your brief verbal presentation of this part of the presentation.

Functional Requirements:

  1. Users must be able to see a combined 1-5 rating for a book from all users

  2. Users must be able to search for books in the system

  3. Users must be able to input their reviews

  4. Users must be able to have their reviews saved and view their previous reviews

  5. Users must be able to mark books as favorited - front end

  6. Users must be able to save books to their profile as favorite reads - back end

  7. Users must be able to sort by good/bad reviews

  8. Users must be able to read other users reviews

  9. Users must be able to add a book to the database if its not in the system

  10. Users must be able to save books to their "to be read" list
